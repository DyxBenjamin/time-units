## Development

```bash
# Install dependencies
bun install

# Build the library
bun run build

# Watch for changes during development
bun run dev

# Format code
bun run format

# Lint code
bun run lint
```

## Publishing

```bash
# Patch version
bun run publish:patch

# Minor version
bun run publish:minor

# Major version
bun run publish:major
```
